# TimeToFate - リアルタイム占いアプリ

時間に基づいてイベントを予測するUnityベースのアプリです。原理は玄学的ですが、アルゴリズムは比較的科学的です。

予測の原理は道教の小六壬（しょうろくじん）に基づいており、時辰起局法を使用しています。従来のように占い師が指で計算する代わりに、このアプリは1回のクリックで自動的に計算を行い、旧暦の月日と現在の時間を手動で判定する手間が省けます。

**時辰起局法**

計算方法：今日が旧暦a月初bで、c番目の時間帯の場合、x=(a+b+c-2)%6

| x    | 1          | 2          | 3          | 4          | 5    | 6          |
| ---- | ---------- | ---------- | ---------- | ---------- | ---- | ---------- |
| 運勢 | 大吉(大安) | 流連(小凶) | 中吉(速喜) | 赤口(中凶) | 小吉 | 空亡(大凶) |

**注意事項**

関連資料を調査する過程で、小六壬のアルゴリズムは統一されていないことが判明しました。道教伝、江氏、邵氏など異なる流派に分かれています。本プロジェクトでは比較的広く採用されていると言われる道教伝のアルゴリズムを使用しています。その正確性については、個人の判断に委ねられます。風水の学問に盲目的に迷信するべきではありませんが、個人の行動や意思決定の参考にすることは許容されます。
